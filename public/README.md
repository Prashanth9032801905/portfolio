# Public Folder Instructions

## Adding Your Profile Image

To add your profile image to the portfolio:

1. **Save your image** as `profile.jpg` in this `public` folder
2. **Recommended dimensions**: 400x400 pixels (square format)
3. **File size**: Keep under 500KB for faster loading
4. **Format**: JPG or PNG

## Current Files

- `profile.jpg` - Your profile image (replace with your actual photo)

## Notes

- The image will automatically appear in the Hero section
- It's displayed in a circular frame with animated background
- The image is optimized for web with Next.js Image component
