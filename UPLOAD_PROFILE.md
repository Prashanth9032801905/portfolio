# How to Upload Your Profile Image

## 📸 Quick Steps:

1. **Locate the public folder:**
   ```
   portfolio-template/
   └── public/
       ├── profile.jpg (replace this)
       ├── profile.svg
       └── README.md
   ```

2. **Prepare your photo:**
   - ✅ Square format (400x400px recommended)
   - ✅ High quality but under 500KB
   - ✅ JPG or PNG format
   - ✅ Professional headshot

3. **Upload your photo:**
   - Delete the current `profile.jpg` file
   - Upload your photo as `profile.jpg` in the `public` folder
   - Make sure the filename is exactly: `profile.jpg`

## 🎯 What You'll See:

Your profile image will appear with:
- 🌟 Glowing background effects
- 🎭 Smooth animations
- 📱 Responsive sizing
- ✨ Professional circular frame
- 🎪 Parallax scroll effects

## 🚨 Troubleshooting:

**Image not showing?**
- Check filename is exactly `profile.jpg`
- Ensure file is in the `public` folder
- Refresh the browser (Ctrl+F5)
- Check browser console for errors

**Image looks distorted?**
- Use a square image (1:1 ratio)
- Recommended size: 400x400 pixels
- Keep file size under 500KB
